Volt Documentation

Installation

Volt provides a main template.xhtml and additional layoutmenu.xhtml, topbar.xhtml files for the base layout. These 3 files must be placed under WEB-INF folder. Provided empty-page.xhtml is a sample content page using the main template.xhtml that defines "content" as the main ui:define placeholder. Other required resources are the css, js, image and font files that are located inside resources/volt-layout folder, simply copy the volt-layout folder to your %WEB-APP-FOLDER%/resources folder so that final path would be %WEB-APP-FOLDER%/volt-layout/. Please refer to demo app or maven project of the demo app as the reference.

Theme is a regular PrimeFaces theme and setup is simple as defining primefaces.THEME context parameter in web.xml as "volt", refer to themes page for more information.

Background Colors

Volt Layout has 4 different background colors, Blue, Leaden, Red and Green. Body given one of four distinct classes , the background color is adjustable.
Class names are, 'BlueBody', 'LeadenBody', 'RedBody', 'GreenBody'

Menu Modes

There are two alternatives for menu orientation, default mode is inline where menu is displayed at the left side of layout and alternative is overlay menu that is enabled by adding "PopupMenu" style class to the body element.

Migration Guide
2.0.1 to 2.1
  • - Replace theme jar with the new jar.
2.0.1 to 2.0.2
  • - Update layout.js, volt-layout.css, volt-layout.less, core-layout.css and template.xhtml
  • - Replace theme jar with the new jar.
  • - Replace VoltMenuRenderer.java with the new VoltMenuRenderer.java.
  • - Replace Volt font files with the new Volt font files.(volt-layout/fonts/*)
2.0 to 2.0.1
  • - Replace theme jar with the new jar.
1.0 to 2.0
  • - Update volt-layout/js/layout.js and volt-layout/css/volt-layout.css.
  • - Replace theme jar with the new jar.
Core Layout

Core Layout is a set of CSS classes used to create responsive grid layouts with additional utilities. Volt uses Core Layout internally, however for grid layout you may also choose another utility like Grid CSS or Bootstrap Grid.
